# Device API
## Advertise
Advertise with the following name.

* format: `<Model number> <Address without :>`
* Model number: `BMZ001<Color>`
* Example: `BMZ001BK 0123456789AB`

Color information is as below.

| Color | Name |
| --- | --- |
| BK | Black |
| NB | Navy Blue |
| PK | Pink |
| PU | Purple |
